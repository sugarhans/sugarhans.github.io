查询集

all filter exclude orderby 调用这些函数会产生一个查询集
特性：

  • 惰性查询
    只有在实际使用数据的是时候才会发生对数据库的真正查询
  • 缓存
    当使用同一个查询集之后再使用这个查询集时，使用的是缓存中的结果

限制查询集：
可以对一个查询集进行取下标和切片操作来限制查询集的记过。对一个查询集进行切片操作会产生一个新的查询集,下标不能为负数

exits() 方法：查询集是否存在
